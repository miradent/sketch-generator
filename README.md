# 数字图像自动生成素描画

基于 Python 的桌面应用，支持从本地图片、摄像头拍照生成素描画，并提供多种风格和参数调节。集成语音搜索和 OCR 文字识别。

## 功能

- 13 种素描算法：高斯模糊、边缘检测、自适应阈值、形态学、双边滤波、细节增强、交叉阴影、拉普拉斯、XDoG、铅笔画、水彩边缘、炭笔画、钢笔画
- 彩绘线条模式、黑白反转等后期处理
- 实时预览和图像保存（支持 PNG/JPG）
- 摄像头拍照、OCR 文字识别（CnOCR + RapidOCR）
- 语音关键词搜索（讯飞听写 API）
- 图像批量搜索与文件夹浏览

## 运行环境

- Windows 10/11
- 摄像头和麦克风（使用 OCR/语音功能时需要）
- 无需安装 Python，下载 `素描生成器.zip` 解压后直接运行 `素描生成器.exe`

## 使用说明

1. 下载 [Releases 中的最新版本](https://github.com/你的用户名/仓库名/releases) 并解压。
2. 双击 `素描生成器.exe` 启动。
3. 通过左侧控件选择图片或拍照。
4. 在算法列表中切换风格，调整参数，点击“生成”。
5. 点击“保存作品”导出结果。
6. 右侧可搜索本地图片或使用语音输入关键词。
7. 点击“OCR扫描”通过摄像头识别文字并搜索。
